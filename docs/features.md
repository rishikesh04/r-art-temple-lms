# Features - R Art Temple LMS

## User Roles

### 1. Admin (Teacher)
Only one admin account should exist for now.

### 2. Student
Students from Class 6 to Class 10.

---

## Access Rules

- Student signup is allowed
- Student accounts should default to `pending`
- Admin can approve or reject students
- Only approved students can access the platform
- No outsider student should be able to use the platform
- No public admin registration should exist

---

## Student Features

- Signup
- Login
- Access after approval
- View dashboard
- View upcoming tests
- Attempt scheduled MCQ tests
- Submit tests
- View results
- View leaderboard
- View analytics
- View weak topics/chapters
- View performance history

---

## Admin Features

- Admin login
- Dashboard
- View pending students
- Approve / reject students
- View all students
- Create and manage questions
- Create and schedule tests
- Assign tests by class and subject
- View submissions
- View student performance
- View test leaderboard
- View analytics

---

## Test Features

- MCQ-based tests
- Class-specific access
- Subject-specific tests
- Chapter-specific tests
- Scheduled availability
- Time-limited attempts
- Auto evaluation
- One attempt per student (MVP)

---

## Leaderboard Features

- Rank by score
- Tie-breaker by time taken
- Secondary tie-breaker by submission time

---

## Analytics Features

### Student Analytics
- Total tests attempted
- Average score
- Best score
- Worst score
- Subject-wise performance
- Chapter-wise weakness
- Score trend over time

### Admin Analytics
- Top performers
- Weak students
- Class-wise trends
- Test performance overview
